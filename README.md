# Run app
* Clone the repository using `git clone https://github.com/NJACKWinterOfCode/SoundHaven.git`
* `cd SoundHaven`
* `npm install`
* `npm run electron`

# Development
* For using the chrome inspector tools uncomment `win.webContents.openDevTools()` in the main.js file.
